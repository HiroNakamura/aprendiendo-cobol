**IF-ELSE-END-IF**

```java
*> INTRODUCIR DATOS CON 'ACCEPT'
 DISPLAY 'INTRODUCE NOMBRE:'
 ACCEPT WS-NOMBRE

   IF WS-NOMBRE EQUAL "MANDRAKE" THEN
      DISPLAY 'BIENVENIDO'WS-NOMBRE
   ELSE
      DISPLAY 'AUN NO TIENES USUARIO REGISTRADO'WS-NOMBRE
   END-IF.
 ```
