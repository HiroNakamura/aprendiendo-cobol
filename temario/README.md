```java

```

**COMPUTE**
```java

*> RESTA

*> PRODUCTO

*> POTENCIACION
COMPUTE WS-RESULTADO = WS-NUM-A ** WS-NUM-B.
*> LONGITUD DE UNA CADENA
COMPUTE WS-LONGITUD = FUNCTION LENGTH(WS-CADENA).
*> REDONDEAR

```

**OPERACIONES ARITMÉTICAS BÁSICAS**
